# House Price Analysis (R Project)

This project is part of my Data Science and Analytics coursework.

## Overview
Analysis of a housing dataset (996 properties) to identify factors influencing house prices and build a prediction model.

## What I did
- Data cleaning and inspection
- Data visualisation (histograms, boxplots, scatter plots)
- Correlation analysis
- Multiple linear regression model
- Model diagnostics (VIF, Cook’s Distance, leverage)
- House price prediction

## Tools
- R (readxl, ggplot2, dplyr, car)

## Files
- R script (analysis + model)
- Report (full explanation)
- Presentation slides
